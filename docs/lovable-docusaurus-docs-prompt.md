# Lovable Docusaurus Docs Prompt

Use this prompt in Lovable to align the no-code demo with the Docusaurus implementation.

```text
Update the Zeza.co.za demo documentation experience so `/docs/` behaves like a Docusaurus-style documentation hub.

Requirements:

1. Replace the static docs landing page with a Docusaurus-inspired documentation hub.
2. Use a left sidebar or mobile drawer navigation.
3. Group documentation into these categories:
   - Resource Assets
   - Products and Courses
   - Tools and Platforms
   - Articles and Education
   - Community and Skool
   - Lead Magnet Operations
   - Launch and Automation
   - Implementation and QA
4. Each Markdown document should have a title, description and clear section headings.
5. Add cards on the docs overview page for the most important docs:
   - Digital Marketing Made Easy Guide
   - Online Business Starter Planner
   - Skool Community Integration
   - Download Funnel Architecture
   - Measurement Plan
   - Content Governance
6. Keep the Zeza brand style:
   - warm cream background
   - deep burgundy accents
   - rose accent colour
   - subtle gold highlights
   - rounded cards
   - clean mobile layout
7. Add a Docs link in the homepage nav and footer.
8. Preserve the Skool group CTA:
   https://www.skool.com/dreamworx-life-3717/about
9. Keep all messaging honest, beginner-friendly and no-hype.
```
