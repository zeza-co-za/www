# Lead Magnet Measurement Plan

## Core metrics

- Landing page visits
- Email form submissions
- Submission conversion rate
- Guide downloads
- Welcome email open rate
- Welcome email click-through rate
- Discovery call bookings
- Community waitlist joins

## Recommended events

- `lead_magnet_view`
- `lead_magnet_email_submit`
- `lead_magnet_download`
- `discovery_call_click`
- `community_waitlist_click`

## Reporting view

Create a weekly report showing:

- Traffic source
- Conversion rate by channel
- Top devices
- Top countries
- Drop-off points
- Follow-up actions generated
