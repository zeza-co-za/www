# Docusaurus Frontmatter Standard

Every Zeza.co.za Markdown document should use Docusaurus-compatible frontmatter where practical.

```yaml
---
id: example-doc
title: Example Doc
description: Short search-friendly description.
sidebar_position: 1
---
```

## Required fields

- `id`
- `title`
- `description`

## Optional fields

- `sidebar_position`
- `slug`
- `tags`
- `keywords`

## Content style

- Use clear H1 and H2 headings.
- Keep the tone beginner-friendly.
- Avoid hype and income guarantees.
- Use checklists and tables where they help the reader act.
