# Downloads

This folder is reserved for Zeza.co.za lead magnets and downloadable digital products.

## Digital Marketing Made Easy Starter Guide

The first lead magnet is the **Digital Marketing Made Easy Starter Guide**.

Recommended production file path:

```text
/downloads/digital-marketing-made-easy-starter-guide.pdf
```

Landing page CTA to wire in production:

```html
<a class="btn btn-primary" href="/downloads/digital-marketing-made-easy-starter-guide.pdf" download>
  Download the Free Guide
</a>
```

The PDF file should be uploaded as a binary asset during the production deployment step or through the hosting provider's static assets upload flow.
